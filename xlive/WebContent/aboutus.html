公司簡介

波峰軟體成立於1992年，早期致力於發展DOS中文GUI（Graphic User Interface）視窗介面及資料庫應用軟體開發。1993年由波峰軟體所研發的 MSDOS 視窗應用系統 (Probe-Windows) 正式與台灣 IBM 簽約為 VLS (Vendor Logo Software) 產品。基於多年作業系統專精與資料庫應用軟體經驗，波峰軟體於1996年投入軟體元件 (Software Component) 的技術開發，歷經數十位系統與資料庫工程師參與，三年的不斷創新設計，一年的資料庫實況測試。1999年八月波峰軟體推出SoftwareLIVE 2000™，是為當時最先進、最快速的資料庫Client-Server前端發展系統。系統以完全視覺化的軟體元件，物件導向（OOP）與事件驅動 (Event Drive) 的程序概念，提供使用者建構Client-Server資料庫應用程式。

SoftwareLIVE 2000™

此革命性的軟體製程系統，改寫過去程式設計及套裝軟體的發展歷程。不但所有人皆能輕易地在Windows上撰寫程式；簡單易學的設計理念，大幅地縮短應用軟體開發及學習的歷程；而其跨越所有資料庫的能力，更將使開發者，在撰寫一版應用程式後，通用於所有資料庫；隨時可按需求無限制地修改、新增應用系統，使得完全為客戶量身訂製的軟體，可以快速地開發設計，成為真正的所謂〝軟體工廠〞。

jLIVE Builder™

2000年波峰軟體以Java™ 跨平台技術全面提升解決方案至多層式網際網路應用 (N-tier Internet Web Application) 。 2001年六月100 % pure Java™ 網際網路應用程式開發平台jLIVE Builder™ 問世，jLIVE Builder™ 提供程式開發者運用Multi-Tier、Component Architecture、Distributed Computing的軟體技術，以高速的生產力，開發 Web-based 的 Java™ 網際網路應用程式。 jLIVE Builder™ 從設計開發流程、程式佈建執行以至系統維護延伸，皆有完整的解決方案可循，專業級的網際網路報表系統更是企業系統應用上不可或缺的工具。

行銷日本

2002年二月 波峰軟體於日本秋越原舉辦jLIVE Builder™ 產品說明會，2002年六月日本 TSNET 株式會社與波峰軟體簽約總代理jLIVE Builder™ 日本行銷，2002年拾月jLIVE Builder™ 正式於日本上市。

放眼世界

jLIVE Builder™ 以其前瞻性的網際網路企業解決方案，先進的軟體開發製程，快速的技術導入，提供E化世代的企業，整合其全國、兩岸三地以及世界佈局所需的應用軟體系統。2003 年波峰軟體 jLIVE Builder™ 中國、歐美行銷已展開積極佈建，屆時jLIVE Builder™ 將成為世界性軟體開發系統，運用 jLIVE Builder™ ，透過網際網路，企業將得以管理全世界。

網際網路的企業應用整合(EAI)

網際網路 (internet) 無遠弗屆地連接起全世界的資訊情報，也串連起世界性的商業市場。網際網路的企業應用整合 (Enterprise Application Integration) 成為企業發展上的必須，如何快速整合新的網際網路方案和舊有應用系統的網際網路化，也成為企業成長的當務之急。jLIVE Builder™ 正是企業架構企業應用整合 (EAI) 系統最佳的工具與環境。2001年八月 國際知名筆記型電腦製造業商，完成以jLIVE Builder™ 建構全球 RMA 系統；2002年元月全國最大電腦製品維修公司，以jLIVE Builder™ 開發完成全國維修 Call Center系統；2003年四月全國最大旅遊網，以jLIVE Builder™ 開發完成旅遊業 ERP系統。

核心技術

波峰軟體全系列產品皆以 100% pure Java™ 開發，適用於各種作業平台。以jLIVE Builder™ 所發展的應用程式亦遵循 J2EE 架構，在企業應用整合 (EAI) 中能與各種 Web 機制緊密地協同合作 (Collaboration) 。

  • 軟體元件技術 (Software Component Technologies)

SoftwareLIVE 2000™ 採用 C++ ，jLIVE Builder™ 以 100 % pure Java™  Swing set 及 MVC 架構開發。軟體元件技術不但是 jLIVE Builder™ 系統基礎，使用 jLIVE Builder™ 所開發的應用程式亦是軟體元件技術的具體實現。

  • RAD開發平台 (Rapid Application Development)

波峰軟體的開發平台，提供程式設計師在完全視覺化的設計環境中，隨時設計，馬上執行，不但使程式的流程更正確，設計的品質亦相對提高。系統以高效率的 Java™ Swing components 元件連結資料，元件內部以 Java™ MVC 技術運作資料。

  • 程式產生器 (Code Generator)

jLIVE Builder™ 是一個 100 % pure Java™ 程式產生器，波峰軟體以軟體元件技術，整合於 RAD 開發平台，讓沒有 Java™ 語言設計經驗的使用者，亦能快速的開發系統及產生程式的原始碼 (Java™ Source Code) 。

  • 資料庫系統 (Database System)

從桌上型資料庫 (dBASE、Access‧‧) 到 SQL-Base (MSSQL、Oracle、DB2‧‧) 以至於 MainFrame 資料庫 (AS400、Fujistu K-serial、SymfoWARE、Cache‧‧)，波峰軟體除了擁有實際應用程式經驗外，亦將其整合至jLIVE Builder™ 中。使用者透過jLIVE Builder™ 將能輕易地存取控管各種類型資料庫系統。

  • 多層式WEB應用程式 (N-tier Web application)

多層式架構在網際網路的企業應用整合環境是絕對必需的，應用程式執行時是不允許客戶端 (client) 直接存取後端的資料庫伺服器。企業中防火牆內的資料庫伺服器與系統資源，必需透過中間層 (middle tier) 取得服務。jLIVE Builder™ 是內設至少三層式的架構，中間層 (middle tier) jLIVE™ middle ware 是 Java™ Servlet ，適用於各種作業平台。

  • 集中式應用程式、分散式運算模組 (Centralized programs & distributed computing)

網際網路應用程式必須分散模組化 (distributed modules)，集中於 Web 伺服器管理，客戶端 (client) 要求服務或執行程式模組時，Web 只需傳輸目前客戶端 (client) 所需服務的小程式模組即可，不需安裝整套的應用程式。jLIVE Builder™ 運用 Java™ Applet 、Swing set 與 class loader 的軟體元件技術，在網際網路應用程式的開發、執行與維護，不但能達到集中式應用程式、分散式運算模組的需求，在整體系統的運算負載 (computing loading) 中，亦達到分散式運算 (distributed computing) ，亦即是系統將使用者的操作需求，分別由客戶端、中間層及伺服器端按最有效率的方式分配擔任運算角色。

  • 網際網路報表系統 (Internet Report System)

jLIVE Builder™ 是目前唯一在程式開發上，提供完整網際網路報表系統解決方案的平台。波峰軟體報表系統是在 Internet 執行列印作業，報表以絕對位置量測列印元件，在報表分頁 (Paging)、按需求列印 (Print On Demand )、列印資料計算與傳輸 (Printing data)、圖檔、套表列印等等的機制上，皆以網際網路列印為考量，達到真正地所見即所得。

  • 國際化 (International)

波峰軟體全系列產品皆為多語系系統，不但可以在各種作業平台設計執行，亦會根據作業平台語系之不同自動登轉。jLIVE Builder™ 的多國語系版本亦提供相同的功能，讓使用者開發的程式在不同語系的客戶端執行時，能自動以客戶端語系顯示。
